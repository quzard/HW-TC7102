# HW-TC7102


## 获取 ip地址

```bash
pip3 install -r requirements.txt

export ROUTER="192.168.1.1"
export PASSWORD="abcd"

python3 hw.py
```

## 腾讯DDNS

```bash
pip3 install -r requirements.txt

export ROUTER="192.168.1.1"
export PASSWORD="abcd"

export DDNSToken="1234"
export DDNSURL="a.com"
export DDNSDomain="sub"

python3 hw.py
```
